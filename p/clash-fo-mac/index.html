Clash for Windows 在 macOS 安装后无法打开，提示损坏

sudo xattr -r -d com.apple.quarantine /Applications/Clash\ for\ Windows.app

Aug 14, 2021
阅读时长: 1 分钟

网络下载应用被 Apple 添加隔离标识，终端执行以下命令解除即可：

1

sudo xattr -r -d com.apple.quarantine /Applications/Clash\ for\ Windows.app

附上完整文档地址：
https://docs.cfw.lbyczf.com/contents/questions.html
